H8427 תָּוָה / תוה

tâvâh

(taw-vaw' | taw-WAW | ta-VA)

Definition

verb; a primitive root; to mark out, i.e., (primitive) scratch or (definite) imprint; scrabble, set (a mark).

  • to scrabble, limit, mark, make or set a mark
    • (Piel) to mark
    • (Hiphil) to set a mark
